Who I Am

I'm Pavan Poriya, currently pursuing my Master of Computer Applications (MCA) in India. My core interests lie at the intersection of data engineering, data analysis, and backend development. I enjoy working with data — cleaning it, transforming it, visualizing it, and building the pipelines that make it useful.

On the backend side, I build APIs and web applications using Python frameworks like FastAPI and Flask. I believe in writing clean, readable code and following solid engineering practices. Every project I take on is an opportunity to learn something new and build something that actually works.
